# Приложение Заметки

### Технологии:
- Backend: Java, Spring
- Frontend: Thymeleaf
- PostgreSQL JPA
- Hibernate
- Maven

### Описание
В этом приложении можно:
- посмотреть текущие заметки
- добавить новую заметку
- изменить заметку
- удалить заметку

### Как запустить
```chmod +x db_init.sh```

```./db_init.sh```

```mvn package```

```java -jar notes-app-1.jar```

Сервис будет доступен по адресу http://localhost:8080/
